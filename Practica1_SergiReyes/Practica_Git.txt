En primer lloc caldrà que us doneu d'alta a GitHub.

    Crea un repositori públic de nom: NomAssignatura_NomCognom
    Crea un fitxer dins d'una carpeta: Practica1_NomCognom/Practica_Git.txt
    El fitxer ha de contenir el dia i la hora a l'instant de fer el commit.

Sincronitzar el github amb l'atom (sudo apt-get install git):

    Crear una carpeta i obrir un terminal dins la carpeta
    Comandes des del terminal:
    git config --global user.name "user_name"
    git config --global user.email "email_id"
    git init
    git remote add origin https://github.com/user_name/Mytest.git
    git pull origin Practica1_NomCognom
    Obrir la carpeta creada com a projecte amb l'atom
    Creem una branca que es digui: Practica1
    Busquem el fitxer que hem creat abans, escrivim el dia i l'hora a continuació del que hem escrit abans
    Obrim el panell de git, afegim el fitxer modificat (stage all), posem un comentari i fem commit
    (Si ens dona error, pot ser que haguem de tornar a escriure:
    git config --global user.name "user_name"
    git config --global user.email "email_id")
    Fem un push
    Anem a la web de GitHub i comprovem que hem actualitzat el fitxer

Escriu l'adreça del repositori per poder-lo corregir.
